# 🌍 BorderCrossInfo - XML Data Engine

**BorderCrossInfo** je ASP.NET MVC aplikacija za praćenje stanja na graničnim prelazima. Podaci se čuvaju i obrađuju putem XML serijalizacije unutar `App_Data` foldera.

## 🚀 Ključne Funkcionalnosti
- **CRUD operacije:** Upravljanje prelazima (dodavanje, prikaz, izmena, brisanje).
- **Pretraga i filtriranje:** Pretraga po nazivu i filtriranje po državi.
- **XML Servis:** Posebna klasa za komunikaciju sa XML bazom podataka.
- **Statistika:** Pregled prosečnog vremena čekanja i opterećenosti po državama.
- **Google Maps:** Integrisan prikaz lokacije prelaza na mapi.

## 🛠️ Tehnologije
- **Framework:** ASP.NET MVC (.NET Framework / VS 2026).
- **Data:** XML (LINQ to XML).
- **Validacija:** Data Annotations i jQuery Unobtrusive Validation.

## 📖 Uputstvo za pokretanje
1. Klonirajte repozitorijum sa GitHuba.
2. Otvorite `.sln` fajl u Visual Studio 2026.
3. Pokrenite **Restore NuGet Packages** (desni klik na Solution).
4. Pritisnite **F5** za pokretanje aplikacije.
